# Portfolio

Web制作の学習成果をまとめたポートフォリオサイトです。

## URL

https://okamura84.github.io/portfolio/

## 概要

これまで制作したJavaScriptアプリを一箇所で確認できるポータルサイトとして制作しました。

## 制作物一覧

- [Piano App](https://github.com/okamura84/piano-app) : Web Audio APIを用いた音声再生アプリ
- [Stopwatch App](https://github.com/okamura84/stopwatch-app) : JavaScriptのタイマー処理を用いた時間計測アプリ

## 使用技術

- HTML5 / CSS3
- JavaScript

## 制作期間

約2か月

## 工夫した点

- Flexboxを用いたレイアウト設計により、画面幅に応じて柔軟に配置が変わる構成にした
- ハンバーガーメニューをJavaScriptで実装し、モバイル環境でも操作しやすくした
- hoverアニメーションを追加し、ユーザーの操作に対する視覚的フィードバックを向上
- 各アプリへの導線を明確にし、迷わずアクセスできるUI設計を意識した

## 今後の課題・改善点

- Reactを用いたアプリケーションの追加
- APIを利用したアプリケーションの追加
- ダークモードへの切り替え機能の実装
- CSSの設計をより整理し、再利用性を高めたい
- アニメーションやインタラクションのバリエーションを増やしたい
